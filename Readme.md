# DriveEase 🚗💨

DriveEase is a modern car rental fleet management and live operational monitoring platform. It provides car rental businesses with real-time tracking, live vehicle dispatch metrics, an automated analytical reporting suite, and an interactive fleet status dashboard.

---

## 🛠️ The Tech Stack

The architecture is built on top of high-performance, industry-standard web engines to optimize data delivery speeds without heavy framework overhead:

*   **Core Backend Processing Engine:** PHP 8.2 (Leveraging dynamic multi-file modular layouts)
*   **Persistent Storage Layout:** MySQL Relational Database Matrix (Optimized schema design)
*   **Live Dashboard Pipeline:** AJAX Asynchronous Polling Loops (Enabling 10-second background fetch sequences)
*   **User Interface Layer:** Vanilla JavaScript (ES6+ standard), Semantic HTML5, and Core CSS Grid/Flexbox Layouts
*   **Version Control & CI/CD DevOps Pipeline:** Git, GitHub Architecture, and Automated GitHub Actions Runner

---

## 🚀 Key Architectural Modules

1.  **Live Fleet Telemetry Core:** Tracks vehicle deployment states instantly using AJAX requests.
2.  **Interactive Performance Metric Indicators:** Aggregates and surfaces key rental analytical parameters dynamically.
3.  **Cross-Origin Modular Matrix System:** Uses strict separation of logic parameters (`includes/db.php`) for easy migration from local testing sandboxes to production cloud servers.

---

## ⚙️ How to Deploy Globally

This repository is configured to replicate modern serverless workflows (like Vercel). Code updates pushed directly to the `main` branch trigger automated GitHub Actions runners to sync and hot-swap code instantly onto the live hosting cPanel engine.
